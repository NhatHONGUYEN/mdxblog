# Une aventure dans le développement web

Mon parcours en tant que développeur web a été une véritable aventure d'apprentissage personnel, débutant avec des vidéos de formation disponibles "from scratch". Dans cet article, je vais évoquer les défis que j'ai dû surmonter, les choix techniques que j'ai effectués, ainsi que les leçons tirées de cette expérience enrichissante.

## Mes débuts en développement web

J'ai commencé mon voyage dans le développement web en visionnant des vidéos de "from scratch" recommandées par un ami, <strong>Maiko</strong>. Au départ, j'ai exploré des ressources gratuites pour assimiler les bases, puis j'ai poursuivi avec des vidéos plus avancées afin d'approfondir mes connaissances. Toutefois, j'ai rapidement réalisé que je peinais à achever un seul exercice sans consulter les solutions proposées.

Malgré ces obstacles, j'ai découvert une forte affinité pour le CSS. J'éprouvais un réel plaisir à concevoir des designs attrayants, ce qui m’a naturellement dirigé vers le développement front-end. J'ai commencé à travailler sur des projets en local, sans les publier, ce qui m'a permis de m'exercer et d'affiner mes compétences.

## Les défis rencontrés

L'apprentissage des langages HTML, CSS et JavaScript a été semé d'embûches. Assimiler les fondamentaux de HTML pour structurer efficacement le contenu, maîtriser CSS afin de styliser les éléments et concevoir des designs attrayants, ainsi qu'apprendre JavaScript pour intégrer interactivité et fonctionnalités dynamiques, étaient autant d'étapes essentielles à franchir.

- **HTML et CSS**
  Apprendre à structurer et styliser efficacement une page web pour répondre aux besoins des utilisateurs.

- **Responsive Design**
  Concevoir des designs harmonieux et adaptatifs pour une expérience utilisateur optimale sur tous les appareils.

- **JavaScript**
  Ajouter de l’interactivité et des fonctionnalités dynamiques pour rendre les applications plus vivantes.

La stylisation avec CSS a également représenté un défi majeur. Apprendre à utiliser les sélecteurs et propriétés CSS pour créer des designs harmonieux, gérer le responsive design pour garantir une expérience utilisateur optimale sur divers appareils, et explorer les animations et transitions CSS pour enrichir l'expérience utilisateur, ont été des compétences cruciales à acquérir.

## En vrai...

C'était peut-être compliqué, mais j'ai quand même ressenti une grande satisfaction personnelle en terminant mon projet. Sans vraiment me douter que ce n'était que le début.
